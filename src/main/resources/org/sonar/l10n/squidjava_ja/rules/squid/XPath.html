このルールを使うと、XPath 表現式を使って、自由に Java のルールを定義することができる。

違反は、XPath の表現式が返す値に基いて決められる。例えば XPath 表現式が:

  • 単一または一連の AST ノードを返す場合、それぞれのノードについて、指定したメッセージを持つ違反が行に対して作られる。
  • 論理値を返す場合、その論理値が真(true)である場合に、指定したメッセージを持つ違反がファイルに対して作られる。
  • それ以外の場合、違反は作られない。

以下に示すのは、各 if 文についての違反を記録するための XPath 表現式である : //ifStatement
